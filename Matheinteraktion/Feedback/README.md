# SolveLab Feedback-Formular (lokal)

Dieses kleine Tool ist eine einzelne HTML-Datei ohne Installation.

## Nutzung

1. `index.html` im Browser oeffnen (Doppelklick reicht).
2. Optional Felder ausfuellen (es gibt keine Pflichtfelder).
3. Auf "Als Markdown speichern" klicken.
4. Die heruntergeladene `.md` per Mail oder Chat senden.

## Was automatisch passiert

- Leere Felder bekommen sinnvolle Defaults (z. B. Tester = Anonym).
- Ticket-IDs werden als `FB-001`, `FB-002`, ... erzeugt.
- Dateiname enthaelt Datum/Uhrzeit, Ticket-Typ und Tester.

## Hinweise

- Fuer mehrere Themen kannst du mit "+ Weiteres Ticket" mehrere Tickets erfassen.
- Optional gibt es einen Button "Markdown kopieren" fuer schnelles Einfuegen.
- Das Tool arbeitet komplett lokal im Browser.
